## 📘 Bootstrap - Estudos e Práticas

Repositório criado para armazenar exercícios, exemplos e práticas utilizando Bootstrap, com foco no aprendizado de criação de interfaces modernas e responsivas para páginas web.

O projeto reúne experimentos e pequenos layouts desenvolvidos durante o processo de aprendizado de Bootstrap aplicado ao desenvolvimento front-end, utilizando HTML, CSS e JavaScript.

O Bootstrap é um framework front-end bastante utilizado para acelerar o desenvolvimento de interfaces web, oferecendo componentes prontos, sistema de grid responsivo e estilos padronizados para elementos HTML.
